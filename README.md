- **前沿技术栈**：Vue3 + Vite + Openlayers 组合，享受极速开发体验

### 环境准备

```bash
Node.js >= 18.0.0
npm >= 9.0.0
```

### 安装依赖

```bash
npm install
```

### 运行开发服务器

```bash
npm run dev
```

### 打包构建

```bash
npm run docs:build
```
